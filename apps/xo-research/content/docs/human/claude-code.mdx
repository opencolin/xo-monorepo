## Higgsfield AI

**Impact:** Generate images and media with Higgsfield AI from inside Claude Code with one slash command.

**How to:**

1. Add the skill: `npx skills add higgsfield-ai/skills --agent claude-code --yes`
2. Install the CLI: `npm install -g @higgsfield/cli`
3. Sign in: `higgsfield auth login`
4. Restart Claude Code so it loads the new skill.
5. Use in Claude Code: `/higgsfield-generate [prompt]`
