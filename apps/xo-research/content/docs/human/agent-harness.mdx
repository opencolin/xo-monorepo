## Scope

This human track focuses on how to design harnesses that make agents reliable across long, multi-step work.

## What this covers

- Context management patterns and compaction strategy
- Single-agent vs multi-agent tradeoffs by task type
- Tooling interface design and failure containment
- Runtime reliability loops (resume, retry, rollback, observability)

## Working method

1. Capture primary source claims.
2. Convert claims into XO-specific implications.
3. Distill into short implementation guidance.
4. Record open questions and test plans.
