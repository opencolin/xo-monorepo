## Scope

This human track defines how XO Swarm should run lightweight agents with strict folder scope and fast trigger loops.

## What this covers

- Swarm role model (observed and observer agents)
- Dispatch topology and folder ownership rules
- Triggering behavior when new information is added
- Safety contracts, logging, and promotion workflow

## Operating goals

- Keep agents small and CLI-first.
- Keep runs auditable and easy to rerun.
- Keep outputs reviewable before promotion.
