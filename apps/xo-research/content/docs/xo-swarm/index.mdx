## What this section is

This section defines how XO Research runs a lightweight swarm of agents that can be triggered quickly and safely.

## Plan

The swarm runs with two role types:

- **Observed agents**: produce output inside one folder scope.
- **Observer agents**: review outputs, detect drift, and propose focused corrections.

The system favors fast loops:

1. Trigger observed agents on scoped tasks.
2. Run observer pass on new outputs.
3. Publish accepted outputs to docs.
4. Repeat with small iterations.

## Operating principles

- One folder per agent scope.
- Small prompts and short runs.
- Plain logs and traceable changelogs.
- No cross-folder edits without explicit handoff.
