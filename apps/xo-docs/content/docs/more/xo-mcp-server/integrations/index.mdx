The XO MCP Server works with any MCP-compatible client. Connect once, then control XO deployments directly from your IDE or AI assistant.

## Supported Clients

- [Cursor](/more/xo-mcp-server/integrations/cursor) — 2-minute setup inside your editor
- [Claude](/more/xo-mcp-server/integrations/claude) — Connect via Claude Web Custom Connectors
- [Claude Code](/more/xo-mcp-server/integrations/claude-code) — Add with a single terminal command
- [ChatGPT](/more/xo-mcp-server/integrations/chatgpt) — Use the XO GPT for knowledge base sharing
- [Devin](/more/xo-mcp-server/integrations/devin) — Configure via Devin's MCP Marketplace
- [Manus](/more/xo-mcp-server/integrations/manus) — Import JSON config in Manus Connectors
